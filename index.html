ABOUT ME

I was first introduced to coding when working for an e commerce company that developed theyre own website. I would observe the developers as they were coding along on their computers and all i could notice at the time was that their monitors look like someting out of the Matrix. Since then i took the time to learn HTML & CSS on my spare time, but defintely felt the need to learn more. This when I took a leap of faith to apply to Galvanize, which is a full stack web dev immersive program. This is where i was able to learn javascript and how servers and apis work. With that knowledge i am now seeking roles as a front end developer, but also ive had the pleasure to work with clients to assist them in creating their online presence wither it be for a personal blog site or import export of products for the business clients.

THINGS IVE WORKED ON

Runners mind

POWERED BY REACT FRONTEND | NODE.JS BACKEND

Social app that gives users the ability to follow eachother and create posts about their running status. Runners are able to follow eachother to recieve updates on runners activities. They can also like and comment on posts.

Runners mind Mobile App

POWERED BY REACT NATIVE FRONTEND | PYTHON BACKEND

Mobile app created to give the ability to runners to be able to track their runs and progress. Using the users location the app tracks distance, time and pace. They can then post their runs and other users can see their progress.

Whatta Meal

POWERED BY HTML | CSS | JAVASCRIPT

Whatta Meal is an app idea that I came up with for the sole purpose to create and save all their reciepes and have one place to be able to check back and cook their most favored reciepes. Future feature will be to give the users the ability to search and see other recipes that they might like and want to save to cook for a later time.

Gmail Clone

POWERED BY REACT

A gmail clone I created using react. Able to create messages with Subjects and Bodies. Also able to star specific messages and place labels for specific groups.

Election dApp

POWERED BY SOLIDITY | HTML | CSS | JAVASCRIPT

Election dApp was something I wanted to play around with using the etherium blockchain. Able to use gas to be able to place votes on whichever the dApp is created for.

Movie Search

POWERED BY REACT

A simple movie search app using react. Hits an api that then returns movies depending on the search criteria. Able to like specific movies. Future feature is to be able to create a list and display all the liked movies.

Spotify Clone

POWERED BY REACT NATIVE

Spotify clone that was created for mobile apps. Inspied by the spotify app I wanted to create it to see what could've been used to create such a sleek and very easy user experience.
